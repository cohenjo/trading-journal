## Three-Stream Sprint Summary

### Stream 1: Issue #341 — Stacked Income Chart Cumulative Bug Fix
- **Shipped:** Commits `1649369` (fix), `cfb3740` (history note)
- **Root cause:** `getOptionsYearlyCashFlow()` used `MAX(cash_flow_cumulative)` per year, treating inception-cumulative column as if it reset annually. Result: 2025 options showed ~$373k instead of actual ~$96k.
- **Fix:** Changed to `SUM(cash_flow_total)` (monthly net) per year, providing true per-year delta.
- **Paired work:** Fenster (chart), McManus (query + data design)
- **Learning:** Cumulative-vs-per-year columns are a common trap in financial time-series; always clarify whether you need cumulative-to-date or per-period delta. Use the right column + right aggregation.
- **Status:** ✅ CLOSED
